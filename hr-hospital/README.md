## Встановлення

### Вимоги

- Odoo 17 Community Edition
- Python 3.10+
- PostgreSQL 14+

### Крок 1: Клонування репозиторію

git clone https://github.com/your-repo/hr-hospital.git

### Крок 4: Встановлення модуля

**Через веб-інтерфейс:**
1. Перейдіть у Налаштування → Модулі → Модулі
2. Знайдіть "Hospital Management"
3. Натисніть "Встановити"

### Встановлення з демо-даними

Для завантаження демо-даних створіть базу з опцією "Load demonstration data":

Демо-дані включають:
- 5 спеціальностей лікарів
- 8 лікарів (включаючи 2 інтернів з менторами)
- 15 пацієнтів з контактними особами
- 12 хвороб з ієрархією (3 рівні)
- 25 візитів різних статусів
- 20 діагнозів
- Розклад для всіх лікарів

### Меню модуля

Після встановлення з'явиться головне меню **"Лікарня"** з підменю:
- Пацієнти
- Контактні особи
- Лікарі
- Спеціальності
- Розклад
- Хвороби
- Візити
- Діагнози

### Типові сценарії

#### Реєстрація нового пацієнта
1. Лікарня → Пацієнти → Створити
2. Заповніть ПІБ, дату народження, контактні дані
3. Призначте персонального лікаря
4. Збережіть

#### Запис на візит
1. Лікарня → Візити → Створити
2. Виберіть пацієнта та лікаря
3. Встановіть дату/час та тип візиту
4. Збережіть

#### Завершення візиту з діагнозом
1. Відкрийте візит
2. Додайте діагноз через вкладку "Діагнози"
3. Натисніть кнопку "Завершити"

#### Генерація розкладу лікаря
1. Лікарня → Лікарі → виберіть лікаря
2. Дія → Згенерувати розклад
3. Вкажіть початок тижня, кількість тижнів, робочі дні
4. Натисніть "Згенерувати"
